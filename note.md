# C-Minus+x编译器项目设计方案

本学期修读编译原理课程,  根据课程各章节内容做出编译器的相应部分, 最终拼接在一起就可以得到完整的C-Minus+x语言编译器. 

C-Minus的完整语法由附录A说明.  x部分的特性和功能待定. 语言源程序文件后缀名设置为`.cmnsx` 

本项目拟使用C++语言进行开发, 使用C++11的特性. 项目将会同步在github平台上, 地址为:
https://github.com/AXILUOWEI/CminusxCompiler
